## Resumen principal
Este proyecto consiste en el diseño y desarrollo de un vehículo a escala motorizado, equipado con una arquitectura de sensores avanzada que incluye dispositivos ultrasónicos, acelerómetros, giroscopios e infrarrojos. La integración de estos componentes tiene como objetivo optimizar los estándares de seguridad, maniobrabilidad y eficiencia operativa del prototipo.
Para garantizar una interacción fluida entre el usuario y la máquina, el sistema incorpora un módulo de comunicación Bluetooth. Este componente actúa como el puente crítico de transferencia de datos, permitiendo la sincronización en tiempo real entre el software de control (aplicación móvil) y el hardware de procesamiento del vehículo.
El propósito central de este proyecto intermodular es promover el aprendizaje dinámico en las áreas de electrónica y telecomunicaciones. Se busca transformar la percepción tradicional del estudio técnico en una experiencia de ocio educativo, orientada especialmente al público juvenil. El fin último es demostrar que el rigor científico y la complejidad tecnológica poseen una vertiente creativa y satisfactoria.
Más allá de la divulgación, el proyecto se enfoca en dos vertientes de análisis:
	
Interacción Usuario-Sistema: Evaluar la capacidad del público para interpretar el funcionamiento del vehículo y gestionar sus parámetros a través de los ajustes configurados en el software.
Validación Técnica: Verificar el correcto desempeño de los sensores y actuadores, asegurando que su respuesta garantice un sistema robusto, seguro y confortable. Esta fiabilidad es fundamental para satisfacer las expectativas tanto de los desarrolladores como de los usuarios finales.

## Executive Summary
This project involves the design and development of a motorized scale vehicle equipped with an advanced sensor architecture, including ultrasonic devices, accelerometers, gyroscopes, and infrared sensors. The integration of these components aims to optimize the prototype’s safety standards, maneuverability, and operational efficiency.
To ensure seamless human-machine interaction, the system incorporates a Bluetooth communication module. This component serves as the critical data transfer bridge, enabling real-time synchronization between the control software (mobile application) and the vehicle’s processing hardware.
The core purpose of this interdisciplinary project is to promote dynamic learning within the fields of electronics and telecommunications. It seeks to transform the traditional perception of technical studies into an "edutainment" experience tailored specifically toward a younger audience. The ultimate goal is to demonstrate that scientific rigor and technological complexity possess a creative and rewarding side.
Beyond outreach, the project focuses on two analytical dimensions:
User-System Interaction: Evaluating the public's ability to interpret the vehicle's operation and manage its parameters through the configurations set in the software.
Technical Validation: Verifying the correct performance of sensors and actuators to ensure a robust, safe, and comfortable system. This reliability is essential to meeting the expectations of both developers and end-users.
